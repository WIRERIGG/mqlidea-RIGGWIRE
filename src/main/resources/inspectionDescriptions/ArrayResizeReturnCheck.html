Reports calls to ArrayResize() whose return value is not checked.

ArrayResize() returns the new array size on success or -1 on failure (e.g., insufficient memory). Ignoring the return value can lead to out-of-bounds access or silent data loss when the resize fails.

Example (bad):

ArrayResize(prices, 1000);
prices[999] = Ask; // May crash if resize failed

Example (good):

if (ArrayResize(prices, 1000) < 0) {
    Print("Failed to resize array");
    return;
}
prices[999] = Ask;
