This inspection reports for or while loops with empty bodies ({}). An empty loop body usually indicates missing implementation or dead code that should be removed.

Example of problematic code:

for(int i = 0; i < OrdersTotal(); i++) {}  // Warning: empty loop body

Add the intended logic or remove the loop:

for(int i = 0; i < OrdersTotal(); i++) {
    if(OrderSelect(i, SELECT_BY_POS))
        ProcessOrder();
}
