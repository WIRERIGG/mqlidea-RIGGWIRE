Reports live prices (Bid/Ask, or MarketInfo() with MODE_BID/MODE_ASK) read inside a loop without RefreshRates() (MQL4).

The predefined Bid and Ask variables are snapshots taken when the event handler started; they do not update by themselves while a loop runs. Call RefreshRates() inside the loop before each price read (typically before retrying a trade operation) so the values reflect the current market.
