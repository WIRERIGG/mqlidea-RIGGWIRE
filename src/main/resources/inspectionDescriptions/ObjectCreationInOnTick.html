Reports object allocations using new inside OnTick() or OnCalculate() event handlers.

Creating objects on the heap in high-frequency handlers causes allocation overhead every tick, leading to performance degradation and potential memory fragmentation.

Example (bad):

void OnTick() {
    CTrade *trade = new CTrade();
    trade.PositionOpen(...);
    delete trade;
}

Example (good):

CTrade trade; // Global or static

void OnTick() {
    trade.PositionOpen(...);
}
