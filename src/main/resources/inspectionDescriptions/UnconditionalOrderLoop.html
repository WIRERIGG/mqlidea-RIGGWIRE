Reports OrdersTotal() or PositionsTotal() calls in OnTick() without any change detection or caching mechanism.

Iterating over all orders/positions on every tick is wasteful when order state rarely changes. Use the OnTrade() or OnTradeTransaction() event handlers for order state changes, or cache the count with a static variable and only re-scan when it changes.

Example (bad):

void OnTick() {
    for (int i = PositionsTotal() - 1; i >= 0; i--) {
        // Check every position on every tick
    }
}

Example (good):

void OnTick() {
    static int prevTotal = 0;
    int total = PositionsTotal();
    if (total == prevTotal) return;
    prevTotal = total;

    for (int i = total - 1; i >= 0; i--) {
        // Only scan when count changes
    }
}
