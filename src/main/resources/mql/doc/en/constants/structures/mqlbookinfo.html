MqlBookInfo

This structure provides information about the market depth data.

struct MqlBookInfo
  {
   ENUM_BOOK_TYPE type;         // Order type from ENUM_BOOK_TYPE
   double         price;        // Price
   long           volume;       // Volume
   double         volume_real;  // Volume with extended accuracy
  };

Note

The MqlBookInfo structure is used to obtain market depth information via the MarketBookGet() function.
