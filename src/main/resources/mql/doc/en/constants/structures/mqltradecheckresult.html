MqlTradeCheckResult

This structure is used to check the result of a trade request before sending it. It is filled by the OrderCheck() function.

struct MqlTradeCheckResult
  {
   uint    retcode;      // Response code
   double  balance;      // Balance after the trade
   double  equity;       // Equity after the trade
   double  profit;       // Floating profit
   double  margin;       // Margin required for the trade
   double  margin_free;  // Free margin after the trade
   double  margin_level; // Margin level after the trade
   string  comment;      // Comment to the response code
  };

Note

MqlTradeCheckResult is used with OrderCheck() to validate a trade request before actually sending it to the server. This allows checking if there are enough funds, whether the parameters are correct, and estimating the margin requirements.
