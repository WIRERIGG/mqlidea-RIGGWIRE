Deal Properties

Constants for obtaining deal properties using HistoryDealGetInteger(), HistoryDealGetDouble(), and HistoryDealGetString().

ENUM_DEAL_PROPERTY_INTEGER

ID

Description

DEAL_TICKET

Deal ticket. Unique number assigned to each deal

DEAL_ORDER

Deal order number

DEAL_TIME

Deal execution time

DEAL_TIME_MSC

Deal execution time in milliseconds since 01.01.1970

DEAL_TYPE

Deal type (ENUM_DEAL_TYPE)

DEAL_ENTRY

Deal entry - entry in, entry out, reverse (ENUM_DEAL_ENTRY)

DEAL_MAGIC

Deal magic number (see OrderSend)

DEAL_REASON

The reason for the deal (ENUM_DEAL_REASON)

DEAL_POSITION_ID

Identifier of the position the deal belongs to

 

ENUM_DEAL_PROPERTY_DOUBLE

ID

Description

DEAL_VOLUME

Deal volume

DEAL_PRICE

Deal price

DEAL_COMMISSION

Deal commission

DEAL_SWAP

Accumulated swap at close

DEAL_PROFIT

Deal profit

DEAL_FEE

Fee charged for a deal

DEAL_SL

Stop Loss level

DEAL_TP

Take Profit level

 

ENUM_DEAL_PROPERTY_STRING

ID

Description

DEAL_SYMBOL

Deal symbol

DEAL_COMMENT

Deal comment

DEAL_EXTERNAL_ID

Deal identifier in an external trading system
