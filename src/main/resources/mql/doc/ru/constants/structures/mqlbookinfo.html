MqlBookInfo

Структура предоставляет информацию о данных стакана цен.

struct MqlBookInfo
  {
   ENUM_BOOK_TYPE type;         // тип заявки из перечисления ENUM_BOOK_TYPE
   double         price;        // цена
   long           volume;       // объём
   double         volume_real;  // объём с повышенной точностью
  };

Примечание

Структура MqlBookInfo используется для получения информации о стакане цен с помощью функции MarketBookGet().
