MqlParam

Структура используется для передачи параметров при создании хэндлов индикаторов с помощью функции IndicatorCreate().

struct MqlParam
  {
   ENUM_DATATYPE type;          // тип входного параметра
   long          integer_value; // поле для хранения целочисленного типа
   double        double_value;  // поле для хранения типа double
   string        string_value;  // поле для хранения строкового типа
  };

Примечание

Поле type указывает, какое из оставшихся полей содержит фактическое значение. Тип может быть одним из значений ENUM_DATATYPE: TYPE_BOOL, TYPE_CHAR, TYPE_UCHAR, TYPE_SHORT, TYPE_USHORT, TYPE_INT, TYPE_UINT, TYPE_LONG, TYPE_ULONG, TYPE_FLOAT, TYPE_DOUBLE, TYPE_STRING, TYPE_DATETIME, TYPE_COLOR.
