MqlTradeCheckResult

Структура используется для проверки результата торгового запроса перед его отправкой. Заполняется функцией OrderCheck().

struct MqlTradeCheckResult
  {
   uint    retcode;      // код ответа
   double  balance;      // баланс после совершения сделки
   double  equity;       // эквити после совершения сделки
   double  profit;       // плавающая прибыль
   double  margin;       // маржа, необходимая для сделки
   double  margin_free;  // свободная маржа после сделки
   double  margin_level; // уровень маржи после сделки
   string  comment;      // комментарий к коду ответа
  };

Примечание

MqlTradeCheckResult используется совместно с OrderCheck() для проверки торгового запроса перед его фактической отправкой на сервер. Это позволяет проверить достаточность средств, корректность параметров и оценить требования к марже.
