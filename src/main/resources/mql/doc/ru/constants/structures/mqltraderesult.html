MqlTradeResult

Структура содержит результат торгового запроса, отправленного функцией OrderSend().

struct MqlTradeResult
  {
   uint     retcode;          // код возврата операции
   ulong    deal;             // тикет сделки, если исполнена
   ulong    order;            // тикет ордера, если выставлен
   double   volume;           // объём сделки, подтверждённый брокером
   double   price;            // цена сделки, подтверждённая брокером
   double   bid;              // текущая цена Bid
   double   ask;              // текущая цена Ask
   string   comment;          // комментарий брокера к операции
   uint     request_id;       // идентификатор запроса, установленный терминалом
   uint     retcode_external; // код возврата внешней торговой системы
  };

Примечание

Результат торгового запроса возвращается функциями OrderSend() и OrderSendAsync(). Поле retcode содержит код возврата торгового сервера, указывающий результат операции.
