MqlTradeTransaction

Структура описывает торговую транзакцию. Передаётся обработчику OnTradeTransaction() при возникновении торгового события.

struct MqlTradeTransaction
  {
   ulong                     deal;            // тикет сделки
   ulong                     order;           // тикет ордера
   string                    symbol;          // символ торговли
   ENUM_TRADE_TRANSACTION_TYPE type;            // тип транзакции
   ENUM_ORDER_TYPE          order_type;      // тип ордера
   ENUM_ORDER_STATE         order_state;     // состояние ордера
   ENUM_DEAL_TYPE           deal_type;       // тип сделки
   ENUM_ORDER_TYPE_TIME     time_type;       // тип истечения ордера
   datetime                 time_expiration; // время истечения ордера
   double                   price;           // цена
   double                   price_trigger;   // цена срабатывания стоп-лимитного ордера
   double                   price_sl;        // уровень Stop Loss
   double                   price_tp;        // уровень Take Profit
   double                   volume;          // объём в лотах
   ulong                     position;        // тикет позиции
   ulong                     position_by;     // тикет встречной позиции
  };

Примечание

Поле type определяет тип транзакции. Набор полей, содержащих значимые данные, зависит от типа транзакции. Например, для TRADE_TRANSACTION_ORDER_ADD заполняются только поля, связанные с ордером.
