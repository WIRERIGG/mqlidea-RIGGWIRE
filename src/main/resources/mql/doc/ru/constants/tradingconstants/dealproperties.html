Свойства сделок

Константы для получения свойств сделок с помощью функций HistoryDealGetInteger(), HistoryDealGetDouble() и HistoryDealGetString().

ENUM_DEAL_PROPERTY_INTEGER

Константа

Описание

DEAL_TICKET

Тикет сделки. Уникальный номер, присваиваемый каждой сделке

DEAL_ORDER

Номер ордера сделки

DEAL_TIME

Время исполнения сделки

DEAL_TIME_MSC

Время исполнения сделки в миллисекундах с 01.01.1970

DEAL_TYPE

Тип сделки (ENUM_DEAL_TYPE)

DEAL_ENTRY

Направление сделки - вход в рынок, выход из рынка, разворот (ENUM_DEAL_ENTRY)

DEAL_MAGIC

Магическое число сделки (см. OrderSend)

DEAL_REASON

Причина совершения сделки (ENUM_DEAL_REASON)

DEAL_POSITION_ID

Идентификатор позиции, к которой относится сделка

 

ENUM_DEAL_PROPERTY_DOUBLE

Константа

Описание

DEAL_VOLUME

Объём сделки

DEAL_PRICE

Цена сделки

DEAL_COMMISSION

Комиссия по сделке

DEAL_SWAP

Накопленный своп при закрытии

DEAL_PROFIT

Прибыль сделки

DEAL_FEE

Сбор, взимаемый за сделку

DEAL_SL

Уровень Stop Loss

DEAL_TP

Уровень Take Profit

 

ENUM_DEAL_PROPERTY_STRING

Константа

Описание

DEAL_SYMBOL

Символ сделки

DEAL_COMMENT

Комментарий к сделке

DEAL_EXTERNAL_ID

Идентификатор сделки во внешней торговой системе
