CopySeries

Получает данные нескольких таймсерий за один вызов для указанного символа и таймфрейма.

int CopySeries(string symbol_name, ENUM_TIMEFRAMES timeframe, int start_pos, int count, ulong rates_mask, ...)

Возвращаемое значение

Возвращает количество скопированных элементов или -1 в случае ошибки.
